# Number List Statistics (Python)

This program asks the user to enter **five numbers**, stores them in a list, and displays:

- The **sum** of all numbers
- The **maximum** value
- The **minimum** value

This exercise reinforces:
- List operations
- For loops
- Input handling
- Built-in Python functions (`sum`, `max`, `min`)

## How to Run

```bash
python3 number_lists_stats.py
